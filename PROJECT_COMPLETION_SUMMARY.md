## 🔧 Customization Guide

### 1. Add Your Profile Picture
- Save image as `assets/prince_profile.png`
- No code changes needed
- Automatically loads on startup

### 2. Change Your Name
```dart
// In UserConfig class (line ~13)
static const String fullName = "Your Name";
static const String nickname = "YourNickname";
```

### 3. Modify Colors
```dart
// Search and replace throughout main.dart
Color(0xFF00D4FF)  → Your color code
```

### 4. Add/Edit Habits
```dart
// In _initializeHabits() method (line ~50)
Habit(
  id: '11',
  name: 'Your New Habit',
  emoji: '🎯',
  completionDays: List.generate(31, (i) => i % 2 == 0),
)
```

### 5. Change Tracking Period
```dart
// In PremiumHeader widget (line ~185)
Text('March 2026 • Tracking Period')
// Change to any period
```
