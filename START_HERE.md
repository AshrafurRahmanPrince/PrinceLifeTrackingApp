## 🔧 Quick Customization

### Add Your Profile Picture
1. Take a photo (square)
2. Save as `assets/prince_profile.png`
3. Restart app
4. Done!

### Change Your Name
1. Open `lib/main.dart`
2. Find `UserConfig` class (line ~13)
3. Edit `fullName` and `nickname`
4. Save and restart

### Change Colors
1. Open `lib/main.dart`
2. Search: `Color(0xFF00D4FF)` (cyan)
3. Replace with your color
4. Save and restart

### Modify Habits
1. Open `lib/main.dart`
2. Find `_initializeHabits()` method
3. Edit the habit list
4. Save and restart
